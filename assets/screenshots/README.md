# Screenshot Capture Guide

Capture the following images after the live Streamlit application is validated:

1. `01_executive_dashboard.png`
   - App title
   - Recommended supplier
   - Award confidence
   - TCO and risk metrics

2. `02_best_value_decision.png`
   - Lowest price vs best value message
   - Executive value breakdown

3. `03_should_cost_tco.png`
   - Packaging should-cost chart
   - Advanced TCO table

4. `04_supplier_scores.png`
   - Supplier RFQ and decision snapshot
   - Risk, ESG, performance, and total score chart

5. `05_allocation_scenarios.png`
   - Supplier allocation pie chart
   - Scenario stress-test results

6. `06_negotiation_playbook.png`
   - Negotiation simulator metrics
   - Generated playbook

7. `07_executive_outputs.png`
   - Executive sourcing memo
   - Supplier clarification email
   - Explainability panel

8. `08_download_center.png`
   - Excel, CSV, TXT, and JSON download buttons

## Capture Standard

- Use desktop-width Streamlit view.
- Keep browser zoom at 90–100%.
- Avoid showing personal browser tabs or account details.
- Use synthetic demo data.
- Ensure charts and tables are fully loaded.
- Crop consistently and keep text readable.
