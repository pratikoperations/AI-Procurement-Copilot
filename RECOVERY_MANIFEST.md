## Recovery Procedure

1. Open the GitHub repository.
2. Read `PROJECT_STATUS.md`.
3. Read `VERSION_MANIFEST.md`.
4. Read `BUILD_HISTORY.md` and `CHANGELOG.md`.
5. Read `DECISION_LOG.md` for frozen decisions.
6. Read `ROADMAP.md` and `PROJECT_BUILD_PLAN.md` for next steps.
7. Continue from the next incomplete build milestone.
